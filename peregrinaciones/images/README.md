# Imágenes de Peregrinaciones

Coloca aquí las imágenes referenciadas en `peregrinaciones.json`.

## Formato recomendado
- **Portadas**: 1200x800px (ratio 3:2), máximo 300KB
- **Galería**: 1920x1080px máximo, máximo 500KB
- Formato: JPG o WebP
- Nombre: usar guiones y minúsculas
