# Sobre Mí - Delegados y Obispo

Esta carpeta contiene la información de cada delegado, el diácono y el obispo.

## Archivos JSON

- **obispo.json** - Información sobre el Obispo de la Diócesis
- **cura-delegado-1.json** - Información del primer Cura Delegado
- **cura-delegado-2.json** - Información del segundo Cura Delegado
- **diacono.json** - Información del Diácono

## Estructura de datos

Cada archivo contiene:
- `id`: Identificador único
- `tipo`: Tipo de persona (obispo, cura-delegado, diacono)
- `nombre`: Nombre completo
- `cargo`: Cargo dentro de la diócesis
- `foto`: Ruta de la imagen de perfil
- `correo`: Email de contacto
- `telefono`: Teléfono de contacto
- `biografia`: Descripción personal y vocacional
- `redes_sociales`: Perfiles de redes sociales
- `mision`: Misión o propósito
- `fecha_ingreso`: Fecha de incorporación
- `notas`: Notas adicionales

## Imágenes

Las imágenes deben colocarse en la carpeta `images/` con los siguientes nombres:
- `obispo.jpg` - Foto del Obispo
- `cura-1.jpg` - Foto del primer cura delegado
- `cura-2.jpg` - Foto del segundo cura delegado
- `diacono.jpg` - Foto del diácono

Reemplaza el contenido de los JSONs con la información real de cada persona.
