# Imágenes de Eventos

Coloca aquí las imágenes referenciadas en `eventos.json`.

## Formato recomendado
- Resolución: 800x600px (ratio 4:3 para cards)
- Tamaño máximo: 200KB por imagen
- Formato: JPG o WebP
- Nombre: usar guiones y minúsculas

## Imágenes esperadas
- `fatima-salida.jpg`
- `convivencia.jpg`
- `retiro-cuaresma.jpg`
- `guadalupe-salida.jpg`
- `vigilia-inmaculada.jpg`
- `pascua-joven.jpg`
- `campamento-verano.jpg`

## Nota
Las imágenes de eventos son opcionales. Si no se proporciona imagen, la app mostrará un color de fondo basado en el tipo de evento.
