# Imágenes de Noticias

Coloca aquí las imágenes referenciadas en `noticias.json`.

## Formato recomendado
- Formato: JPG o WebP para mejor compresión
- Resolución máxima: 1920x1080px
- Tamaño máximo: 500KB por imagen
- Nombre: usar guiones y minúsculas (ej: `fatima-2025-inscripciones.jpg`)
