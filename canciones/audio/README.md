# Audio de Canciones

Coloca aquí los archivos de audio referenciados en `canciones.json`.

## Formato recomendado
- Formato: MP3 (128-192kbps) para buen balance calidad/tamaño
- Tamaño máximo: 8MB por archivo
- Nombre: usar guiones y minúsculas (ej: `caminando-juntos.mp3`)

## Archivos esperados
- `caminando-juntos.mp3`
- `tu-presencia-es-todo.mp3`
- `aqui-estoy-senor.mp3`
- `ave-maria-guadalupe.mp3`
- `juntos-como-hermanos.mp3`
- `vivo-para-amarte.mp3`
- `peregrinando-fatima.mp3`

## Nota
Si no tienes los archivos de audio, la app mostrará las letras sin la opción de reproducir.
